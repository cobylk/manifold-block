Borrowing some inspiration from [this paper](https://www.jmlr.org/papers/v21/20-345.html) but in the spirit of making things interpretable by design, we attempted to explicitly represent the manifold that data lies on, as it is transformed on its way through a neural network. 
See `Explicit_manfiold_neural_block_math.pdf` for the mathematical ideas and `atlas_autoencoder.py` for the code.

This is a toy repo that stores some quick experiments we did as an extension of a SPAR project; here to showcase the implementation/math for portfolio purposes.
